回収選択デモ 0

CSV推奨。KML/GeoJSONは簡易対応（ポイントのみ）。
GPXのtrkpt/rtept、GeoJSONのLineString、CSVはLatitude/Longitude列に対応。
出力: 日時, 地点名, 現在地緯度, 現在地経度, 車両ID, 運転者
ヘルプ
  • マーカーをタップすると色が切替（黄色/赤）。
  • 「回収」は選択中の地点をCSVに追記。
  • 「外部ナビ（回収済み→次）」は collection_log の最終行と地点CSVの Description から次地点を判定。
  • 位置情報が取れない場合は https かつサイトの位置情報を「許可」にしてください。
